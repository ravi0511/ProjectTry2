# Quickstart: Public Digital Resume Webpage

## Prerequisites

- Node.js 20 or newer
- npm

## Local Development

1. Open the project root.
2. Install dependencies:
   - `npm install`
3. Start the development server:
   - `npm run dev`
4. Open the local URL shown in the terminal.

## Production Build

1. Run:
   - `npm run build`
2. Deploy the generated `dist` folder to any static hosting platform such as GitHub Pages, Netlify, Vercel, or Azure Static Web Apps.

## Expected Result

The resume page should load as a polished, responsive single-page site with a header, summary, skills section, work history, and technologies section.
