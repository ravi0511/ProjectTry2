# Resume Site Frontend

This project contains a public, responsive single-page resume experience built with React, Vite, and Tailwind CSS.

## Development

- Install dependencies: `npm install`
- Start the dev server: `npm run dev`
- Build for production: `npm run build`

## Content Updates

Update the resume details in [src/data/resumeContent.js](src/data/resumeContent.js) to change the public content without modifying the layout structure.
